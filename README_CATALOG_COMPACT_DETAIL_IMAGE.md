# Catalog Compact Detail/Image Update

- Catalog result list no longer shows item images.
- Catalog list rows now show only item name and category/subcategory.
- Removed tier/rarity/stack/status labels from list results.
- Double-clicking an item loads the right-side detail panel.
- The detail panel shows the item image, metadata, description, and crafting requirements.
- Main catalog remains search/category driven and lazy loads compact rows in batches.
